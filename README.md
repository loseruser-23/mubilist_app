# MubieList

merupakan Final Project Kelas Flutter Batch 8

## Spesifikasi

1. Aplikasi ini untuk menampilkan film-film rekomendasi yang membantu user mencari film terbaik yang bisa di tonton di Bioskop
2. Memiliki 3 menu di Navbar, (Search, Home dan Profile)
3. Pada AppBar terdapat API cuaca dari API Publik Weather API berfungsi untuk memberikan informasi cuaca, kalau - kalau user mau keluar lihat prediksi cuaca dahulu.
4. Navbar Search menggunakan Api dari TMDB yang merupakan API publik untuk rekomendasi movie terbaru.
5. Navbar Home pada bagian paling atas terdapat icon dummy untuk melihat menu2 rekomendasi berupa pencarian Box Office, pemesanan tiket dan lokasi-lokasi bioskop terdekat dan lain lain, pada bagian tengahnya ada film-film yang akan tayang dan best offers namun itu juga dummy belum memiliki API, pada bagian bawah terdapat Latest News dengan menggunakan News API yang berhubungan dengan berita mengenai Film-film..
6. Navbar Profile seperti biasa menampikan profil pemilik akun yang rencananya mau menampilkan isi database sang pemilik akun.
7. kemudian di AppBar terdapat logo account dan notifikasi, yang dimana IconButton account mengarahkan pada halaman login yang authentikasinya menggunakan Firebase Auth,
